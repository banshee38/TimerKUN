こちらはタイマーアプリになります。
基本的には半角で数字を入力して頂き、
スタートを押していただくことでカウントダウンされます。

入力した数字＋登録名を入れていただき、
登録ボタンを押していただくことでいつでも登録した物を呼び出せます。
呼び出した物を選んでいる状態で削除ボタンを押して頂くことで、
登録しているセットを削除することができます。
（登録又は削除した際は、反映させるのにタイマーくんを再起動する必要があります。）

また、起動時はボタンを反映させるのに初回のみ2度押して頂く必要があります。
（登録しているセットが一つの場合、初回起動で読み込めないので2種類登録していただく必要があります。）
